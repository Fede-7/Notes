# Proprietà della varianza sotto trasformazioni lineari

Sia Y = aX + b una trasformazione lineare di X (a e b costanti). Come si trasforma la varianza?

**Risultato:** Var(Y) = a² · Var(X). La varianza è moltiplicata per il **quadrato** del coefficiente di scala, non il coefficiente stesso.

**Derivazione:** E[Y] = E[aX + b] = aE[X] + b = aμ_X + b (linearità del valore atteso). Quindi:

Var(Y) = E[(Y - E[Y])²] = E[(aX + b - aμ_X - b)²] = E[(a(X - μ_X))²] = E[a²(X - μ_X)²] = a² E[(X - μ_X)²] = a² Var(X)

**Interpretazione:** La costante additiva b **non cambia la varianza** perché traslare tutti i dati della stessa quantità non modifica la loro dispersione relativa. Moltiplicare per a scala la dispersione per il fattore a². Se a = 2, la dispersione quadruplica; se a = 1/2, la dispersione si riduce a 1/4.

**Applicazione pratica:** Se X è il prezzo in euro con varianza σ² euro², e voglio convertire in dollari (Y = 1.10X), la nuova varianza è 1.10² · σ² = 1.21σ² dollari². Questo modello universale di trasformazioni lineari è cruciale in ingegneria e finanza quando si cambiano unità di misura o si scalano dati.

# Sequenze binarie con k uni: multinomiale

Una **sequenza binaria** di lunghezza n è una successione di n simboli ciascuno 0 o 1. Il numero di sequenze binarie di lunghezza n contenenti esattamente k uni (e quindi n-k zeri) è C(n,k): devo scegliere quali k posizioni su n ricevono l'1, il resto riceve 0.

Questo modello si generalizza al **coefficiente multinomiale**: il numero di modi di partire un insieme di n elementi in sottoinsiemi di cardinalità n₁, n₂, ..., n_m (con Σn_i = n) è n! / (n₁! n₂! ... n_m!). Per il caso binario (m=2) si riduce a C(n,k) = n!/(k!(n-k)!).

Le sequenze binarie con k uni ricorrono frequentemente: conteggio di successi in n prove indipendenti, configurazioni di bit in parole di lunghezza fissa, allocazioni di oggetti indistinguibili in categorie. Il coefficiente multinomiale fornisce il conteggio diretto senza dover iterare sulle scelte successive, rendendolo efficiente computazionalmente nel calcolo probabilistico.

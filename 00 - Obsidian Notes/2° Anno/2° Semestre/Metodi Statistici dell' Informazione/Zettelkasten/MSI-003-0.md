# Valore atteso: il baricentro della distribuzione

Il **valore atteso** (o **media**, o **speranza matematica**) è definito come E[X] = ∑ₖ aₖ · P(X = aₖ) per una variabile aleatoria discreta X con alfabeto 𝒜 = {a₁, a₂, ...}. È una **media pesata** dove i pesi sono le probabilità: i valori con probabilità più alta contribuiscono di più al valore atteso. Fisicamente, rappresenta il **baricentro** della distribuzione di probabilità: se si immaginano pesi infinitesimali (le probabilità) posizionati sui punti dell'asse reale (i valori dell'alfabeto), il baricentro cade dove c'è più massa.

La motivazione viene dalla **legge dei grandi numeri**: se si ripete l'esperimento n volte e si calcola la media aritmetica delle osservazioni x̄ₙ = (1/n)∑ᵢ₌₁ⁿ X(ωᵢ), questa converge al valore atteso quando n → ∞. Formalmente, le frequenze relative fₙ(aₖ) = Nₖ/n convergono alle probabilità P(X = aₖ), quindi x̄ₙ → ∑aₖ · P(X = aₖ) = E[X]. Il valore atteso è quindi il **limite teorico** della media empirica: dice il "valore medio" che ci aspettiamo di osservare su tempi lunghi.

È fondamentale distinguere il valore atteso dalla media aritmetica dell'alfabeto. Quest'ultima pesa tutti i valori ugualmente; il valore atteso è una media ponderata dalle probabilità reali.

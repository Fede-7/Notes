# Entropia di Shannon

Poiché $I(X=x) = -\log_2 p_X(x)$ è essa stessa una variabile aleatoria, l'**entropia** è la sua media:

$$H(X) = -\sum_{x \in \mathcal{X}} p_X(x) \log_2 p_X(x) = E\!\left[\log_2 \frac{1}{p_X(X)}\right]$$

$H(X)$ risponde a: quanti bit in media occorrono per codificare il valore di $X$? L'entropia è massima per la distribuzione uniforme ($H = \log_2 |\mathcal{X}|$ bit) ed è zero per una variabile deterministica. Per una bernoulliana vale al più 1 bit, raggiunto solo con $p = 1/2$.

L'applicazione pratica è nella compressione: un file compresso ideale è una sequenza binaria in cui 0 e 1 sono equiprobabili e indipendenti, cioè con entropia massima di 1 bit per simbolo.

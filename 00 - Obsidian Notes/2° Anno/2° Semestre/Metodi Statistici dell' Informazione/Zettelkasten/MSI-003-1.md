# Legge dei grandi numeri: perché la media campionaria converge

La **legge dei grandi numeri** afferma che la media campionaria **converge** al valore atteso quando il numero di osservazioni cresce. Sia X una variabile aleatoria con valore atteso E[X] = μ. Se si conducono n esperimenti indipendenti e si osservano X₁, X₂, ..., Xₙ, la media campionaria è x̄ₙ = (1/n)∑ᵢ₌₁ⁿ Xᵢ. Allora lim_{n→∞} x̄ₙ = μ con probabilità 1 (convergenza quasi certa).

**Derivazione intuitiva:** raggruppare le osservazioni per valore. Se l'alfabeto è {a₁, ..., aₘ}, il valore a_k appare Nₖ volte in n prove. Allora x̄ₙ = (1/n)∑ₖ Nₖ·aₖ = ∑ₖ (Nₖ/n)·aₖ = ∑ₖ fₙ(aₖ)·aₖ, dove fₙ(aₖ) è la frequenza relativa. Per la definizione di probabilità come limite della frequenza, fₙ(aₖ) → P(X = aₖ) quando n → ∞. Quindi x̄ₙ → ∑ₖ P(X = aₖ)·aₖ = E[X].

La legge dei grandi numeri **non è un assioma** ma una conseguenza della struttura probabilistica. Collega la teoria (il valore atteso) all'esperienza empirica (la media campionaria). Senza questa legge, le distribuzioni di probabilità sarebbero mere astrazioni matematiche senza contatto con la realtà osservabile.

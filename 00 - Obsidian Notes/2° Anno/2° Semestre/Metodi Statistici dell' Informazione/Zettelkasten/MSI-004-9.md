# Correlazione: covarianza normalizzata

La **correlazione di Pearson** (o coefficiente di correlazione) tra X e Y è definita come:

ρ(X, Y) = Cov(X, Y) / (σ_X · σ_Y)

dove σ_X e σ_Y sono le deviazioni standard. È la **covarianza normalizzata** per le scale dei dati, rendendola adimensionale.

**Proprietà fondamentale:** -1 ≤ ρ(X, Y) ≤ 1. Il valore 1 rappresenta correlazione positiva perfetta (Y cresce linearmente con X con coefficiente positivo), -1 rappresenta correlazione negativa perfetta, 0 rappresenta assenza di correlazione lineare.

**Interpretazione:**
- ρ ≈ 1: fortissima relazione positiva
- ρ ≈ -1: fortissima relazione negativa
- ρ ≈ 0: nessuna relazione lineare

**Attenzione:** ρ = 0 non implica indipendenza! Esempio: X ~ Uniforme su [-1, 1], Y = X². Allora E[XY] = E[X³] = 0, E[X] = 0, quindi Cov(X, Y) = 0 e ρ = 0. Eppure Y dipende perfettamente da X (è deterministica). La correlazione cattura solo **relazioni lineari**, non dipendenze generali.

La correlazione è invariante sotto trasformazioni lineari: ρ(aX + b, cY + d) = ρ(X, Y) (cambiare scala o traslare non modifica la correlazione). Questo la rende una misura **robusta** di associazione lineare, indipendente dalle unità di misura.

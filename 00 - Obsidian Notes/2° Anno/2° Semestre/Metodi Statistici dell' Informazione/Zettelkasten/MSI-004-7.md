# Teorema fondamentale: media di una trasformazione

**Teorema:** Data X con PMF p_X(x) e una funzione g, il valore atteso di Y = g(X) è:

**E[g(X)] = ∑_x g(x) · p_X(x)**

Questo risultato è **fondamentale** perché permette di calcolare E[Y] **senza costruire esplicitamente** la PMF di Y. Non serve raggruppare i valori in 𝒴 o preoccuparsi del collasso: si lavora direttamente sull'alfabeto di X, calcolando il valore atteso ponderato di g(x).

**Derivazione (caso molti-a-uno):** Per definizione, E[Y] = ∑_y y · p_Y(y). Poiché p_Y(y) = ∑_{x: g(x)=y} p_X(x):

E[Y] = ∑_y y · ∑_{x: g(x)=y} p_X(x) = ∑_y ∑_{x: g(x)=y} g(x) · p_X(x) = ∑_x g(x) · p_X(x)

(nel penultimo passaggio si usa g(x) = y poiché x mappa a y).

**Applicazione pratica:** Se X = rendita annuale e g(X) = tasse = 0.20·X, allora E[tasse] = E[0.20X] = 0.20·E[X], senza calcolare la PMF delle tasse. Se g(X) = X², calcolo E[X²] = ∑_x x² · p_X(x) direttamente, non ricostruendo la distribuzione di X².

Questo teorema è il ponte tra teoria (PMF di X) e pratica (proprietà di funzioni di X).

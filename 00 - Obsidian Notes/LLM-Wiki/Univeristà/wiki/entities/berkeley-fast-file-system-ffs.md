# Berkeley Fast File System (FFS)
Il **Berkeley Fast File System (FFS)** è il file system che ha introdotto il concetto di vicinanza tra inode e dati attraverso l'organizzazione in gruppi (block groups), migliorando la località dei dati sul disco.

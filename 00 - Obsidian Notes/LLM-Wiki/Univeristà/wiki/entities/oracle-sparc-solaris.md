# Oracle SPARC Solaris

**Oracle SPARC Solaris** è un sistema operativo a 64 bit noto per l'integrazione stretta tra hardware e software. È citato come esempio di architettura che utilizza buffer di storage specifici per le traduzioni (TSB) per gestire la memoria in modo efficiente.

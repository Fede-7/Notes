# EXT4

**EXT4** è il file system moderno di default per molte distribuzioni Linux.

### Caratteristiche principali
- Supporta file fino a 16 TiB e file system fino a 1 EiB.
- Supporta l'allocazione multiblocco.
- Utilizza inode più grandi (256 byte).
- Eredita le funzionalità di journaling e block groups da EXT3.

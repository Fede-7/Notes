# Visual Studio Code

[[visual-studio-code]] è un editor di codice consigliato per lo sviluppo in ambiente [[wsl]]. Viene spesso integrato per fornire un'esperienza di sviluppo fluida tra l'interfaccia grafica di Windows e il terminale Linux.

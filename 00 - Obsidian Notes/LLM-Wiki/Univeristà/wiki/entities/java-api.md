# Java API

La **Java API** è l'interfaccia di alto livello per il linguaggio Java, costituita dall'insieme di classi e interfacce fornite per la programmazione. Essa aggiunge uno strato di astrazione significativo sopra le chiamate di sistema, rendendo il codice indipendente dalle specifiche del sistema operativo sottostante.

Attraverso la Java API, i programmi possono interagire con la **Java Virtual Machine (JVM)** e, indirettamente, con il sistema operativo e l'hardware. Questo livello di astrazione è superiore a quello fornito dalle API di sistema come POSIX o Win32 API.

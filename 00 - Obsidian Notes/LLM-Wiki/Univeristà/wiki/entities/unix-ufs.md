# UNIX UFS

L'**UNIX UFS** (Unix File System) è uno schema di file system che combina diverse tecniche di allocazione per bilanciare prestazioni e utilizzo dello spazio.

### Caratteristiche
- Utilizza una combinazione di allocazione indicizzata e contigua.
- È il predecessore dei moderni file system Linux come la serie EXT.

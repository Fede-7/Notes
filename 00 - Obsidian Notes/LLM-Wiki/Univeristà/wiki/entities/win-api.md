# Win API
La **Win API** è l'interfaccia di programmazione delle applicazioni (API) specifica per il sistema operativo Windows. Fornisce un insieme di funzioni in linguaggio C (contenute in librerie come `kernel32.dll` e `User32.dll`) per interagire con il sistema operativo e le risorse hardware.

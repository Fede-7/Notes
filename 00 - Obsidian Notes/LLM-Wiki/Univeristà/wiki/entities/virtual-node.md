# Virtual Node
Il concetto di **Virtual Node** si riferisce alla virtualizzazione dell'inode per generalizzare le operazioni tra diversi sistemi operativi. Permette di presentare un'interfaccia uniforme per le operazioni sui file indipendentemente dalla struttura specifica del file system sottostante.

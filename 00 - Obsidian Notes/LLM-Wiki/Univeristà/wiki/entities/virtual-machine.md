# virtual machine

Una `virtual machine` (VM) è un sistema che fornisce una separazione completa dell'hardware e del sistema operativo. Ogni VM esegue un proprio sistema operativo guest sopra un hypervisor, risultando più pesante della containerizzazione poiché replica l'intero stack hardware e software.

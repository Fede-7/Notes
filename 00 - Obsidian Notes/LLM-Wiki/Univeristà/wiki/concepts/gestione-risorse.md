# Gestione delle Risorse

Il Sistema Operativo agisce come gestore delle risorse del sistema, assicurando che ogni processo riceva ciò di cui ha bisogno per funzionare e che le risorse vengano rilasciate correttamente alla terminazione.

Le risorse principali gestite includono:
- CPU
- Memoria
- I/O
- File
- Dati di inizializzazione

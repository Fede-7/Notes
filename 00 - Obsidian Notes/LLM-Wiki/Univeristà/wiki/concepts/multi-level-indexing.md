# Multi-level Indexing
Il **multi-level indexing** è una struttura gerarchica di indici (simile alle tabelle delle pagine della memoria virtuale) che permette di indirizzare file di dimensioni molto grandi, utilizzando indici singoli, doppi o tripli.
